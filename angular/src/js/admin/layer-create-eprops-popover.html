The properties below represent the superset of all available properties across all features within your new layer. Displayed values are pulled from first feature to have the given property set.

Properties flagged with the icon have values that are unique across all features within your new layer.

If a property has an explicit type (e.g. string,number or boolean) then this indicates the property is available on all features and has a consistent type across them.

A property type of mixed almost certainly indicates that property in question is either unset or is null for some features.
